# Inventory Utils

A lightweight forge 1.8.9 mod which adds various inventory utilities.

## Features:

### Automatically clicking features:
- `Shift + Left Click + Drag`: Move full stacks of items by dragging over inventory slots
- `Shift + Right Click + Drag`: Move all items except one from stacks (leave one item behind)
- `Ctrl + Left Click + Drag`: Move single items from multiple stacks by dragging over slots

### Item dropping features:
- `Ctrl + Q`: Drop entire stack you're hovering over
- `Ctrl + Shift + Q`: Drop all items of the same type from inventory

All features are customizable through mod options.
